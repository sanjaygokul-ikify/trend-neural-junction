# RFCs
This directory contains Requests for Comments (RFCs) related to the trend-neural-junction project.
Each RFC is stored in a separate file and is assigned a unique number.
# How to propose an RFC
1. Create a new file in the `docs/rfcs` directory with a unique number and a descriptive title.
2. Write a clear and concise proposal in the file.
3. Open a pull request to propose the RFC.
